La place des produits dans la sous alimentation

Pour mieux comprendre nos donnes, nous allons raliser une ACP " Analyse en composantes principales". Nous allons regarder pour "chaque" produit qu'est ce qui le caractrise. LAnalyse en composantes principales, est une mthode consistant a transformer des variables corrles en variables dcorrles. Le principe de cette mthode est de transformer une variable en individus ( par exemple la variable Area est reprsent par des individus ( pays )). Pour nous montrer graphiquement vers quelles variables tend chaque pays laide dun cercle de corrlation.
