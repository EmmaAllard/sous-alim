La place des produits dans la sous alimentation

Pour mieux comprendre nos données, nous allons réaliser une ACP " Analyse en composantes principales". Nous allons regarder pour "chaque" produit qu'est ce qui le caractérise. L’Analyse en composantes principales, est une méthode consistant a transformer des variables corrélées en variables décorrélées. Le principe de cette méthode est de transformer une variable en « individus » ( par exemple la variable ‘Area’ est représenté par des individus ( pays )). Pour nous montrer graphiquement vers quelles variables tend chaque pays à l’aide d’un cercle de corrélation.

Quels facteurs influents la sous alimentation ?

Pour mieux comprendre la sous alimentation et l'expliquer, nous créons un modèle linéaire en prenant compte les corrélations, les premières analyses mais aussi ce que nous savons sur le sujet.

En statistiques, le modèle de régression linéaire est un modèle pour prédire et expliquer une variable quantitative ( pour nous ce sera la sous alimentation ). L’objectif est d’établir un lien entre la sous alimentation et les autres variables. Pour voir l’influence des variables, si elles augmentent ou diminuent la sous alimentation.
